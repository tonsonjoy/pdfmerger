Platform Note
Privacy First
100% client-side — no servers involved

When you drop a file onto EassyOffice, it is read directly from your local filesystem into memory. Processing is performed using pdf-lib — a pure-JavaScript PDF library that runs entirely inside your browser tab. The finished file is handed straight back as a download. No bytes travel over the network at any point.

This architecture makes EassyOffice well-suited for confidential documents — legal contracts, medical records, financial reports — where uploading to a third-party server is simply not an option.

🧠
In-browser processing
pdf-lib runs in JavaScript — no server, no plugin, no extension needed.
📂
Local file access only
Files are read via the File API — browser policy blocks any outbound transfer.
🚫
No file analytics
We have zero visibility into filenames, content, or page counts — nothing is logged.
🔌
Offline capable
After the first page load, all tools work without an active internet connection.
