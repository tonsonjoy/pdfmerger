🎯 Detection midpoint: 10:30 — punch-in before this = Morning, after = Evening
How shift detection works:
The system calculates the midpoint between the Morning and Evening start times. Each row's punch-in time is compared against that midpoint to classify it automatically.

Example: Morning starts 09:00, Evening starts 12:00 → midpoint is 10:30. A punch-in at 09:05 = Morning. A punch-in at 12:10 = Evening. Same employee can have both shifts in the same month.
